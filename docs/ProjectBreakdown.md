# Linux Device Manager - 项目拆分与知识点分析

**版本**: v1.0  
**日期**: 2026-04-18  
**作者**: 开发团队

---

## 一、项目整体架构拆分

```
Linux Device Manager (主项目)
│
├── 模块1: 设备连接管理器 (Device Connector)
│   └── 独立功能: SSH连接、认证、会话管理
│
├── 模块2: 系统监控面板 (System Monitor)
│   └── 独立功能: CPU/内存/磁盘监控、进程管理
│
├── 模块3: 文件传输工具 (File Transfer)
│   └── 独立功能: SFTP文件浏览、上传下载
│
├── 模块4: 远程终端 (Remote Terminal)
│   └── 独立功能: SSH终端、命令历史、快捷指令
│
├── 模块5: 自动化测试平台 (Test Platform)
│   └── 独立功能: 测试用例管理、执行、报告生成
│
└── 模块6: 通用UI组件库 (UI Components)
    └── 独立功能: 可复用的自定义控件、主题系统
```

## 二、各模块详细拆分与知识点分析

---

### 模块1: 设备连接管理器 (Device Connector)

#### 业务边界
- 管理SSH连接的生命周期
- 处理多种认证方式（密码/密钥）
- 维护连接状态
- 提供统一的命令执行接口

#### 核心技术知识点

| 知识点 | 说明 | 应用场景 | 技术细节 |
|--------|------|----------|----------|
| **libssh2库使用** | SSH协议C库封装 | 建立SSH连接、执行命令、文件传输 | 初始化libssh2、创建session、握手认证 |
| **会话状态机** | 管理连接的各种状态 | 连接中、已连接、断开、错误状态切换 | 使用枚举定义状态，状态转换函数 |
| **非阻塞IO** | 异步网络操作 | 连接过程不阻塞UI | libssh2_session_set_blocking(0) |
| **密钥认证** | RSA/ED25519私钥认证 | 免密码登录 | libssh2_userauth_publickey_fromfile |
| **连接池** | 管理多个连接 | 同时连接多台设备 | QVector管理多个SSHClient实例 |
| **心跳检测** | 保持连接活性 | 定时发送保活包 | libssh2_keepalive_send |

### 模块2: 系统监控面板 (System Monitor)

#### 业务边界
- 实时采集系统性能数据
- 可视化展示监控指标
- 进程列表管理与操作

#### 核心技术知识点

| 知识点 | 说明 | 应用场景 | 技术细节 |
|--------|------|----------|----------|
| **QCustomPlot** | 第三方图表库 | CPU/内存实时曲线 | 添加QCPGraph、设置数据点、自动缩放 |
| **定时器机制** | QTimer周期性触发 | 定时采集系统数据 | QTimer::singleShot, start(1000) |
| **数据解析** | 解析/proc文件系统 | 解析CPU、内存、进程信息 | 正则表达式提取数字 |
| **进程信号** | 向进程发送信号 | 终止进程 | kill -9 对应SIGKILL |
| **Model/View** | 进程列表展示 | QTableView显示进程 | QStandardItemModel或自定义Model |
| **数据缓存** | 历史数据存储 | 显示历史曲线 | QVector缓存最近N个数据点 |

#### /proc文件系统解析知识

| 文件路径 | 内容说明 | 解析方法 |
|----------|----------|----------|
| /proc/stat | CPU统计信息 | 计算jiffies差值 |
| /proc/meminfo | 内存信息 | 读取MemTotal/MemAvailable |
| /proc/diskstats | 磁盘IO | 计算读写扇区数 |
| /proc/net/dev | 网络流量 | 解析eth0的收发字节 |
| /proc/[pid]/stat | 单个进程信息 | 提取PID、状态、CPU时间 |

### 模块3: 文件传输工具 (File Transfer)

#### 业务边界
- 远程文件系统浏览
- 文件上传下载
- 文件权限管理

#### 核心技术知识点

| 知识点 | 说明 | 应用场景 | 技术细节 |
|--------|------|----------|----------|
| **SFTP协议** | SSH文件传输 | 文件上传下载 | libssh2_sftp_init, libssh2_sftp_open |
| **文件树Model** | 远程文件系统模型 | QTreeView展示目录 | 实现QAbstractItemModel |
| **异步传输** | 大文件不阻塞UI | 文件传输后台进行 | 在独立线程中传输 |
| **进度回调** | 显示传输进度 | 进度条更新 | libssh2_session_callback_set |
| **文件权限** | Unix权限管理 | 修改文件权限 | chmod, chown命令 |
| **拖拽操作** | 拖拽上传下载 | 从本地拖到远程 | 实现dragEnterEvent, dropEvent |

|--------|------|----------|----------|

## 七、模块接口设计建议

### 模块1: DeviceConnector 对外接口

```cpp
class IDeviceConnector {
public:
    virtual ~IDeviceConnector() = default;
    
    // 连接管理
    virtual bool connect(const ConnectionConfig& config) = 0;
    virtual void disconnect() = 0;
    virtual bool isConnected() const = 0;
    
    // 命令执行
    virtual QString executeCommand(const QString& command, int timeout = 30000) = 0;
    
    // 文件传输
    virtual bool uploadFile(const QString& localPath, const QString& remotePath) = 0;
    virtual bool downloadFile(const QString& remotePath, const QString& localPath) = 0;
    
    // 信号
    virtual void connectionStateChanged(bool connected) = 0;
    virtual void errorOccurred(const QString& error) = 0;
};
```
